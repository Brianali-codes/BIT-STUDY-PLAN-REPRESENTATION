BIT Study Plan: 24-Week Journey

Week 1: Programming Basics

Building the foundation with variables, data types, operators and control flow structures.

Variables Control Flow Data Types

Week 2: Advanced Python

Elevating skills with functions, modules, file operations and object-oriented programming.

Functions OOP Modules

Week 3: Data Structures & Algorithms

Mastering the building blocks of efficient code with lists, stacks, queues and search algorithms.

Lists Algorithms Big O

Week 4: Databases

Understanding data persistence through SQL, normalization, and CRUD operations.

SQL Normalization CRUD

Week 5: Web Development

Creating web experiences with HTML, CSS, and JavaScript fundamentals.

HTML CSS JavaScript

Week 6: Front-end Development

Building interactive UIs with React, state management, and API integration.

React State APIs

Week 7: Back-end Development

Developing server-side applications with Node.js, Express, and REST APIs.

Node.js Express REST

Week 8: Networking

Understanding digital communication through OSI model, protocols, and subnetting.

OSI Model TCP/IP Subnetting

Week 9: Cybersecurity Basics

Protecting digital assets with encryption, hashing, and ethical hacking principles.

Encryption Hashing Ethical Hacking

Week 14: Machine Learning Basics

Teaching computers to learn with supervised and unsupervised learning techniques.

Supervised Unsupervised Model Training

Week 21: Mobile App Development

Creating apps for mobile platforms using React Native, Flutter, and native Android.

React Native Flutter Android

Week 24: Capstone Project

Culmination of learning through development of a full-stack or AI-based project.

Design Implementation Presentation

Weekly Learning Structure

Monday-Tuesday: Concept Introduction

  • Theoretical learning of fundamental principles
  • Educational videos and documentation reading
  • Note-taking on key concepts

Wednesday-Thursday: Practical Implementation

  • Hands-on coding exercises and lab work
  • Problem-solving challenges
  • Mini-projects implementing concepts

Friday: Review and Advancement

  • Quiz or assessment on week's material
  • Review of challenging concepts
  • Preview of next week's material

Weekend: Project Work and Consolidation

  • Mini-project application of weekly learning
  • Review of notes and supplementary reading
  • Study group discussions and networking
